# 乒乓学球体会碰撞流程

当用户发送乒乓球相关的训练体会、技术要点、比赛感悟时，执行以下碰撞补充流程：

## 流程

1. **记录到 obsidian**：保存到 `生活/YYYY-MM-DD-乒乓球训练.md`
2. **更新技术要点框架**：在 `乒乓球/2026-03-22-乒乓球技术要点汇总.md` 中新增条目（如果该要点尚未被收录）
3. **碰撞技术图谱**：将用户体会与 GitHub Pages 技术图谱 `projects/project-table-tennis/index.html` 中现有卡片进行碰撞比对
4. **补充图谱**：如果碰撞发现新的、有价值的洞察，在图谱页面新增对应卡片（使用现有卡片格式：card-header → details → core-text → process-content 各 section）
5. **推送部署**：git push 到 regeorge.github.io

## 技术图谱现有卡片清单（截至 2026-04-19）

1. 击球点技术 · 力量传导（第三点撞击集中向前，第四点放慢往上合力）
2. 身体感知 · 力量传导（重心挂在腿上，不对抗重力）
3. 底层发力 · 力量传导（胯先顶出去，延迟释放）
4. 动作框架 · 结构层（放弃控制权，先走完框架再发力）
5. 迎前意识 · 战术层（击球窗在右前方40-50cm）
6. 击球时机 · 动态层（上升期借力、高点期发力、下降期摩擦）
7. 错题集 · 芒格思维（下旋/上旋/步法/手肘四大误区）
8. 进阶逻辑 · 复利飞轮（滑梯=自动化，复利=积累增长）
9. 正手攻拉 · 撞击优先（亮拍撞实再收压弧线，重心前脚掌）
10. 正手细节 · 踩实与释放（每球踩实顶住，下旋后原地轻颠）
11. 反手定位 · 稳住为攻（反手注意力放右侧后前对住，为正手创造空间）
12. 正手细节 · 引拍与收臂（大臂贴身→小臂打开→看到胶皮黑面）
13. 身体感知 · 雨刮器思维（核心稳定+轻微转腰+放松收小臂）
14. 反手进阶 · 近台退台（近台仅手臂，退台加胯；骑着球往前）
